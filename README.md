### AI-Shield Agentic Framework (v5) 🛡️

**AI-Shield Agentic** — это десктопная система (GUI) на Python для проведения автоматизированного адаптивного аудита безопасности больших языковых моделей (LLM) в соответствии с международным стандартом **OWASP Top 10 for LLM Applications**.

Инструмент реализует концепцию **Multi-turn Adaptive Attacker Agent** (многошаговый ИИ-хакер) и разделяет процесс на поток рассуждений агента (`<thought>`) и саму атаку (`<attack>`).

### 🌟 Ключевые возможности (Core Features)

*   **Agentic Multi-Turn Attacks:** Симуляция живого диалога хакера с моделью-целью (до 8 раундов). Хакер адаптирует промпты на основе ответов жертвы.
*   **Thought Stream & Live Sandbox:** Раздельные экраны в GUI для отслеживания внутренней логики рассуждений ИИ-хакера и живого чата с целевой моделью.
*   **Hybrid Connectivity:** Поддержка локального тестирования через **Ollama** (Llama-3, Mistral) без утечки данных, а также облачных API (OpenRouter, Groq, Anthropic, OpenAI, Azure).
*   **Rule-Based Attack Scorer:** Высокоэффективный модуль оценки успешности взлома (анализ PII-паттернов, утечек кода и обхода фильтров отказа).
*   **SQLite Persistence & Analytics:** Встроенная база данных для сохранения истории аудитов, расчета среднего балла риска и покрытия уязвимостей.
*   **Enterprise Reporting:** Экспорт детализированных отчетов в форматы Markdown и PDF.
*   **Multilingual UI:** Полная поддержка локализации интерфейса (English, Русский, Deutsch).

### 🛠️ Архитектура системы

Проект построен на базе многопоточной архитектуры (`threading` + `queue`), что предотвращает зависание графического интерфейса при длительных сетевых запросах к моделям:

1.  `AIShieldGUI (customtkinter)` — реактивный графический интерфейс.
2.  `LLMClient` — фабрика провайдеров для унификации работы с локальными и облачными API.
3.  `AttackScorer` — семантический анализатор уязвимостей.
4.  `SQLite Storage` — модуль аналитики и сохранения логов сессии.
